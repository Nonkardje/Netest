# Netest - Test d'Addiction à Internet (IAT)

Ce projet est une implémentation web autonome du test IAT (Internet Addiction Test) du Dr. Kimberly Young.

Il permet aux utilisateurs d'évaluer leur niveau de dépendance numérique via un questionnaire interactif de 20 questions, en toute confidentialité (les données restent locales).
